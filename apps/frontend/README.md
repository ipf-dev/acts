# Frontend

This app is the default UI surface for `acts`.

## Structure

- `src/pages` for route-level files
- `src/dashboard-api` for HTTP boundaries
- `src/components/ui` for shared UI primitives
- `src/lib` for helpers and hooks
- `.agents/skills` for frontend-specific agent guidance

## Commands

- `npm install`
- `npm run lint:file-suffixes`
- `npm run dev`
