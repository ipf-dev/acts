ACTS AI Contents Tech Studio
ACTS MANUAL

ACTS는 지금은 에셋 관리 시스템이지만,
앞으로는 AI Contents Tech Studio로 확장됩니다.

ACTS는 하모니 힐즈를 포함한 다양한 콘텐츠 자산을 한 곳에 모아 찾기 쉽게 관리하는 시스템으로 시작합니다. 앞으로는 여러 팀이 AI를 활용해 다양한 콘텐츠를 제작하고, 정리하고, 배포하는 일까지 더 쉽게 할 수 있도록 돕는 스튜디오로 발전해 나갈 예정입니다.

버전 1.3 내부 사용자용 2026년 4월

1. ACTS 소개

ACTS를 처음 접하는 분도 이해할 수 있도록, 무엇을 하는 시스템인지 먼저 간단히 설명드립니다.

ACTS는 무엇인가요?

ACTS는 이미지, 영상, 오디오, 문서, 외부 링크 같은 콘텐츠 자료를 한 곳에 모아 관리하는 내부 시스템입니다. 필요한 자료를 찾고, 정리하고, 다시 활용하기 쉽게 만드는 것이 가장 큰 목적입니다.

쉽게 말해, 팀마다 흩어져 있던 파일과 자료를 한 곳에서 관리하고 함께 쓰기 위한 공간이라고 보시면 됩니다.

ACTS의 방향 ACTS는 단순한 자료 보관함에 머무르지 않습니다. 앞으로는 하모니 힐즈 콘텐츠를 중심으로 여러 팀이 AI를 활용해 콘텐츠를 만들고, 가공하고, 배포하는 과정까지 지원하는 AI Contents Tech Studio로 확장될 예정입니다.

ACTS에서 할 수 있는 일

자료 모으기

이미지, 영상, 오디오, 문서, 링크를 업로드하여 중앙에서 관리할 수 있습니다.

자료 찾기

검색어와 필터를 사용해 필요한 자료를 빠르게 찾을 수 있습니다.

자료 정리하기

태그와 설명을 붙여 자료의 성격을 더 쉽게 구분할 수 있습니다.

함께 활용하기

같은 자료를 여러 팀이 공통 기준으로 참고하고 재사용할 수 있습니다.

이력 확인하기

누가 무엇을 수정했는지 확인할 수 있어 관리가 더 명확해집니다.

안전하게 운영하기

권한에 따라 접근 범위를 나누어 내부 자료를 보다 안전하게 다룹니다.

지원하는 자료 유형

유형설명예시
IMAGE정적 이미지 파일배경 이미지, 캐릭터 시트, 시안, 썸네일
VIDEO영상 파일원본 영상, 편집본, 최종본

OTHER기타 파일ZIP, PSD, AI 등 특수 형식

2. 시작하기

처음 접속했을 때 무엇을 확인해야 하는지 순서대로 안내드립니다.

접속 방법

  • ACTS는 웹 브라우저에서 사용하는 내부 시스템입니다.
  • Chrome 사용을 권장합니다.
  • 회사 Google 계정으로만 로그인할 수 있습니다.
  • 개인 Google 계정이나 외부 계정으로는 로그인할 수 없습니다.
로그인 안내 ACTS는 회사 Google 계정 전용 내부 시스템입니다. 로그인되지 않거나 접근이 되지 않는 경우에는 먼저 회사 계정으로 로그인되어 있는지 확인해 주세요.
처음 사용하는 분께 ACTS는 기능이 많아 보여도 기본 사용 흐름은 매우 단순합니다. 자료 보기 → 검색하기 → 업로드하기 → 상세 정보 확인하기 순서만 익히셔도 대부분의 업무를 진행하실 수 있습니다.

로그인 후 가장 먼저 볼 화면

왼쪽 메뉴

에셋 라이브러리, 관리자 메뉴 등 주요 화면으로 이동하는 곳입니다.

상단 사용자 영역

현재 계정, 역할, 로그아웃 정보를 확인하는 곳입니다.

메인 목록 영역

업로드된 자료를 카드나 목록 형태로 확인하는 중심 화면입니다.

검색 및 필터 영역

원하는 자료를 빠르게 찾기 위해 검색어와 조건을 설정하는 영역입니다.

처음 로그인했을 때 확인할 것

  1. 에셋 라이브러리가 정상적으로 보이는지 확인합니다.
  2. 검색창에 간단한 키워드를 넣어 검색이 되는지 확인합니다.
  3. 내 계정의 역할이 일반 사용자인지 관리자인지 확인합니다.
  4. 권한이 부족하거나 화면이 이상하면 관리자에게 문의합니다.

3. 에셋 라이브러리

ACTS의 중심 화면입니다. 저장된 자료를 확인하고, 찾고, 상세 페이지로 이동할 수 있습니다.

자료를 보는 방식

그리드 보기

썸네일 중심으로 자료를 한눈에 보고 싶을 때 적합합니다. 이미지나 영상 확인에 특히 편리합니다.

리스트 보기

제목, 유형, 등록자, 등록일 등을 비교하면서 보고 싶을 때 적합합니다.

검색하는 방법

검색창에 단어를 입력하면 제목, 설명, 태그를 기준으로 관련 자료를 찾아줍니다. 정확한 이름이 기억나지 않아도 관련 키워드만 입력해 보시면 도움이 됩니다.

검색 팁 자료 이름보다 캐릭터명, 장소명, 주제어로 찾는 것이 더 잘 맞는 경우가 많습니다. 예를 들어 “Coco”, “forest”, “chant”처럼 검색해 보시기 바랍니다.

필터로 더 정확하게 찾기

필터 종류무엇을 고를 수 있나요?언제 쓰면 좋나요?
에셋 유형이미지, 영상, 오디오, 문서, 링크, 기타원하는 자료 형태가 분명할 때
조직소속 조직 기준특정 팀 자료만 보고 싶을 때
업로더등록한 사용자 기준누가 올린 자료인지 알고 있을 때
유형별 상세 조건아트 스타일, 제작 단계, 문서 종류 등비슷한 자료 중 원하는 상태만 보고 싶을 때

상세 페이지로 이동하기

카드나 목록의 한 줄을 클릭하면 해당 자료의 상세 페이지로 이동합니다. 여기서 미리보기, 설명, 태그, 다운로드, 수정 가능 여부 등을 확인할 수 있습니다.

4. 업로드하기

파일 자체를 올릴 수도 있고, 외부 링크만 등록할 수도 있습니다.

업로드의 두 가지 방식

파일 업로드

이미지, 영상, 오디오, 문서 등의 실제 파일을 ACTS에 저장합니다.

링크 등록

Google Drive, YouTube, Notion 같은 외부 주소를 자료처럼 등록합니다.

파일 업로드 순서

  1. 업로드 버튼을 누릅니다.
  2. 파일을 끌어다 놓거나 선택합니다.
  3. 제목과 설명이 필요하면 입력합니다.
  4. 시스템이 자동으로 유형을 잡아주지만, 필요하면 수정합니다.
  5. 태그를 추가합니다.
  6. 업로드를 시작합니다.

6. 태그 사용하기

태그는 ACTS를 더 잘 쓰기 위한 핵심 요소입니다. 자료를 잘 찾게 해주는 ‘검색용 라벨’이라고 생각하시면 됩니다.

태그 입력은 선택이지만 권장됩니다 캐릭터, 장소, 키워드 태그는 필수 입력 항목은 아닙니다. 다만 자료의 성격에 맞게 입력해 두시면 나중에 검색과 재활용이 훨씬 쉬워집니다.

태그는 왜 중요한가요?

파일명만으로는 자료를 찾기 어려운 경우가 많습니다. 태그를 잘 달아두면 나중에 다른 팀도 자료를 쉽게 찾고, 같은 자료를 반복 제작하는 일을 줄일 수 있습니다.

주요 태그 종류

캐릭터 태그

예: Coco, Tony, Polly, Becky. 특정 캐릭터와 관련된 자료를 묶어서 찾을 때 유용합니다.

장소 태그

예: forest, classroom, beach. 배경이나 장면 중심으로 자료를 찾을 때 유용합니다.

키워드 태그

키워드 태그는 자유롭게 입력할 수 있습니다. 다만 너무 새로운 표현을 만들기보다, 자주 쓰이는 공통 키워드를 활용하시면 더 좋은 검색 결과를 만들 수 있습니다.

캐릭터 태그 안내 현재 정책상 캐릭터 태그에는 기존 캐릭터 Coco, Tony, Polly, Becky와 앞으로 추가될 4명의 신규 캐릭터만 등록되어 있습니다. 다른 캐릭터 태그가 필요하시면 관리자에게 말씀해 주세요. 캐릭터 태그는 관리자만 추가하거나 수정할 수 있도록 구성되어 있습니다.
검색은 걱정하지 않으셔도 됩니다 캐릭터 검색은 한국어와 영어 이름을 어느 정도 자연스럽게 찾을 수 있도록 구성되어 있습니다. 예를 들어 Coco로 검색해도 코코 자료를 찾을 수 있습니다.

태그를 잘 다는 기준

  • 캐릭터, 장소, 키워드 태그는 필수 입력 항목은 아니지만, 자료의 성격에 맞게 입력해 두시는 것을 권장합니다.
  • 다른 사람도 바로 이해할 수 있는 단어를 사용해 주세요.
  • 너무 많은 태그를 붙이기보다 꼭 필요한 태그를 중심으로 입력해 주세요.
  • 비슷한 표현이 여러 개 생기지 않도록 기존에 자주 쓰이는 태그를 먼저 확인해 주세요.
키워드 태그 입력 안내 키워드 태그는 자유롭게 입력하실 수 있습니다. 다만 ACTS에서는 자주 사용되는 키워드가 자동으로 제안되기 때문에, 너무 유니크한 키워드를 새로 만들기보다 공통으로 자주 쓰이는 키워드를 사용하는 것이 더 좋은 검색 결과를 만드는 데 도움이 됩니다.

8. 권한과 역할

모든 사용자가 같은 기능을 보는 것은 아닙니다. 역할에 따라 보이는 메뉴와 가능한 작업이 다를 수 있습니다.

기본 역할

역할가능한 일주요 대상
일반 사용자자료 보기, 검색, 업로드, 상세 정보 확인대부분의 내부 사용자
관리자일반 기능 + 사용자/태그/정책/로그 관리운영 담당자

권한이 없을 때는 어떻게 하나요?

버튼이 보이지 않거나 수정이 되지 않는 경우에는 시스템 오류가 아니라 권한 차이일 수 있습니다. 필요한 작업이 있다면 관리자에게 요청해 주시기 바랍니다.

9. 자주 묻는 질문

Q. 제목을 꼭 입력해야 하나요?

A. 꼭 입력하지 않아도 되는 경우가 있지만, 다른 팀이 나중에 찾기 쉽도록 이해하기 쉬운 제목을 넣는 것을 권장합니다.

Q. 같은 파일을 여러 번 올리면 어떻게 되나요?

A. 일부 중복은 자동으로 걸러질 수 있습니다. 다만 비슷한 파일이 많다면 제목과 설명, 태그를 명확히 적어 구분해 주시는 것이 좋습니다.

Q. 태그를 잘못 넣었어요.

A. 수정 권한이 있으면 상세 페이지에서 수정할 수 있습니다. 권한이 없다면 관리자에게 요청해 주시기 바랍니다.

Q. ACTS는 앞으로 무엇이 더 추가되나요?

A. 현재는 에셋 관리가 중심이지만, 앞으로는 여러 팀이 AI를 활용해 콘텐츠를 제작하고 배포하는 과정까지 더 폭넓게 지원하는 방향으로 확장될 예정입니다.
